# programa-o_em_python_senai

Este repositório contém atividades de aprendizado em Python, realizadas como parte do curso de Desenvolvimento de Sistemas no Senai. Durante o curso, estou aprendendo os conceitos fundamentais de programação e como aplicá-los na prática através de diversos exercícios.

O que estou aprendendo:

Tipos de Variáveis: Compreendendo os diferentes tipos de dados que o Python suporta, como inteiros, floats, strings e booleanos, e como utilizá-los em operações.

Operações Aritméticas: Realizando cálculos matemáticos e manipulando números através de operações básicas, como soma, subtração, multiplicação, divisão, entre outras.

Manipulação de Entradas de Usuário: Utilizando a função input() para interagir com o usuário e capturar dados que serão processados pelo programa.

Estruturas de Controle e Fluxo: Aplicando estruturas como condições if/else e laços de repetição para controlar o fluxo de execução do código.

Sistema de Cadastro Simples: Criando um sistema simples de cadastro de usuários, coletando dados como nome, idade e e-mail, e os exibindo de forma organizada.

Essas atividades são um passo importante para solidificar meu aprendizado de programação e começar a entender como construir sistemas e resolver problemas reais utilizando Python.
